Capa de red






Si tenemos en cuenta el modelo OSI, la tercera capa es la "capa de red".

Esta capa es la encargada de proporcionar conectividad y selección de ruta entre
dos sistemas de hosts que pueden estar ubicados en redes geográficamente distintas.

Para ello, utiliza cuatro procesos básicos:

-Direccionamiento
-Encapsulamiento
-Enrutamiento
-Desencapsulamiento






Un buen ejemplo de esta acción sería intentar mandar un archivo de un ordenador a otro.

Independientemente de otras capas como la de enlace de datos, es necesaria ésta para
poder transmitir la información, especificando parámetros como la ruta, el destino...
